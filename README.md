# 📱 WhatsApp Chat Analyzer

## 📌 Project Overview

The WhatsApp Chat Analyzer is a data analytics project built using Python that analyzes exported WhatsApp chat files and generates meaningful insights through statistics and visualizations. The project helps users understand communication patterns, user activity, message trends, emoji usage, and overall chat behavior.

## 🚀 Features

- Upload and analyze exported WhatsApp chat files (.txt)
- Display total messages sent
- Count total words used
- Calculate media files shared
- Count links shared
- Identify the most active users in group chats
- Generate monthly and daily message timelines
- Analyze weekly and monthly activity
- Visualize busiest days and busiest months
- Create activity heatmaps
- Generate WordCloud of the most frequently used words
- Display the most common words
- Analyze emoji usage and frequency
- Interactive charts and graphs for better visualization

## 📂 Project Workflow

1. Export WhatsApp chat from the mobile application.
2. Upload the exported `.txt` file.
3. Preprocess the chat data.
4. Clean and structure the dataset.
5. Perform data analysis.
6. Generate statistics and visualizations.
7. Display insights through an interactive Streamlit interface.

## ▶️ How to Run

1. Clone the repository

```bash

```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```

4. Upload an exported WhatsApp chat file and explore the analytics.
